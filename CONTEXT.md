- **核心用户**：广创部（帮助甲方客户做社媒内容运营的团队）

## 现实业务流程

```
关键词 → 搜索热点 → 甲方提供素材 → 出方案（数字人/口播文案/分镜头/图片图标）
→ 生成口播数字人视频 / 视频素材 → 成片 → 发布 → 评论维护/养号
→ 每月数据回收 → 复盘看板
```

---

## 系统架构：两大模块

### A · 客户中心（一次性建档，不频繁变动）
- **客户档案**：职业、方向、期望等背景
- **账号管理**：一客户可绑定多个账号，每个账号有独立的平台、关键词、风格定位，AI 生成账号规划方案
- **数字人管理**：同一客户可克隆多个形象（上传视频 + 音频），接口由用户自行对接，系统只做 UI 层
- **项目列表**：在客户详情页新建项目

### B · 创作工作台（每个项目循环执行）
- **来源层**（三选一，非强制）：热点抓取新闻信息流 / 甲方文案上传优化 / 关键词 DeepResearch
- **方案层**：AI 生成文字方案，用户确认后自动填入生产模块
- **生产层**（始终可直接使用，不被前置步骤锁定）：
  - 口播文案（含数字人形象 + 音色选择）
  - 分镜头脚本（可增删改，自由添加新镜头，输入提示词即可）
  - AI 图片（含垫图上传功能）
  - AI 图标/贴纸
  - ~~数字人视频~~（已合并入口播文案 tab）

## 核心设计原则

- 初版单一角色，不做任务分配
- 导航逻辑：在客户页建项目，在创作工作台管项目（类似 Linear 的 Project + My Issues）
- 生产工具不被前置步骤锁定，用户可以自由跳入任意模块
- 视觉风格：极简奢品调性，浅色为主，凸显科技感
  - 主色：`#1A1D2E`（深海军蓝）
  - 金色点缀：`#C9A96E`
  - 背景：`#F7F6F3` / `#FAFAF8`
  - 边框：`#E8E6E0`

## 关键交互细节

### 分镜头脚本
- 每条镜头显示：序号、时间段、场景标题、台词/旁白、镜头提示
- 每条镜头可点击编辑所有字段（含提示词）
- 每条镜头有删除按钮
- 底部有「+ 新增镜头」区域，输入提示词即可新增
- 底部有「从口播稿重新拆分」按钮

### AI 图片 · 垫图功能
- 顶部切换：「AI 生成」/ 「上传垫图」
- 上传垫图模式：拖拽区 + 快捷标签（产品白底图 / 生活场景图 / 人物形象图）
- 选择垫图后自动切回 AI 生成模式，并在输入框上方显示已选垫图信息

### 口播文案 · 数字人整合
- 数字人形象选择（展示客户绑定的数字人形象卡片）
- 音色选择（下拉或卡片形式）
- 合并在同一 tab，生成视频按钮也在此处
