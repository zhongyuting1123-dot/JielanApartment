## 核心模块

### 1. 客户中心（客户管理）

- **客户管理**：客户基本信息、所属行业、运营平台、客户画像（目标受众/运营目标/内容调性/规避方向）
- **账号管理**：每个客户下多个平台账号（抖音/小红书/视频号/快手/B站/TikTok），账号有风格、关键词等属性，支持编辑和删除
- **数字人管理**：上传视频+音频克隆数字人形象，系统只做 UI 层
- **AI 运营方案**：基于客户信息自动生成整体运营策略（内容矩阵、发布节奏、增长路径）
- **AI 账号规划方案**：基于账号信息生成账号级别的内容策略

### 2. 创作工作台（数字人视频）

- **项目管理**：新建项目弹窗选择客户→选择账号，项目状态流转（草稿→生产中→已完成→已发布）
- **来源层**（Tab 1）：三选一（热点抓取 / 甲方文案 / 深度调研）
- **方案层**（Tab 2）：业务结合方式（强关联/软植入/不结合）+ 目标时长（15-120秒）→ AI 一键生成完整方案 JSON
- **生产层**（Tab 3）：方案确认后自动填充4个子 Tab
  - 口播稿（含数字人形象+音色选择+视频版本历史）
  - 发布文案（标题+正文+标签）
  - 分镜头（镜头卡片+AI生图prompt+运镜）
  - 素材图片（封面图+配图）
  - 项目主题字段（AI 填充，可手动编辑）
- **布局**：Tab 切换（非瀑布流）

### 3. AI 创作（原 AI 工具箱）

App 式卡片入口，5个子功能，统一布局：左栏表单 + 右栏预览墙

- **商品图生成**：上传1张产品图 → AI 识别商品信息 → 生成1-8张（默认7张）亚马逊规范商品图
- **图生图**：上传参考图 + 提示词 → AI 生成目标场景图片
- **图生视频**：上传1-2张参考图 + 分镜设置 + 提示词 → AI 生成短视频，支持 DeepThink 自动拆分镜头提示词
- **爆款视频复刻**：上传爆款视频 → AI 分析结构 → 用你的产品替换 → 生成同款结构新视频（参考 AdsTurbo Ad Clone）
- **提示词库**：统一管理提示词，三级隔离（系统预设/团队共享/我的模版），支持分类、搜索、变量替换

## 提示词体系（6套）

| 编号 | 名称 | 触发位置 | 输入 | 输出格式 |
|------|------|---------|------|---------|
| 1 | 客户运营策略 | 客户中心→AI运营方案 | 客户信息+平台列表 | Markdown |
| 2 | 单条视频方案 | 创作工作台→方案层 | 客户+账号+来源+业务结合+时长 | JSON |
| 3 | 口播稿 | 生产层→口播稿Tab→重新生成 | 方案JSON+客户信息 | 纯文本 |
| 4 | 发布文案 | 生产层→发布文案Tab→重新生成 | 方案JSON+口播稿+平台规则 | JSON |
| 5 | 分镜头 | 生产层→分镜头Tab→重新生成 | 方案JSON+口播稿+时长 | JSON数组 |
| 6 | 素材图片 | 生产层→素材图片Tab→重新生成 | 方案JSON+封面比例 | JSON数组 |

**数据流**：方案层一次性生成完整 JSON → 自动填充4个 Tab → 用户不满意时单独调用对应提示词重新生成。口播稿必须先于分镜头和发布文案生成。

**提示词文件**：`studio/src/data/systemPrompts.js`
