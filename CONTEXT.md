## 核心模块

### 1. 客户管理（客户中心）

- **客户信息**：品牌名称、所属行业、运营平台、内容方向、建档时间
- **客户画像**：目标受众、运营目标、内容调性、规避方向（可编辑）
- **AI 运营方案**：基于客户信息 + 平台列表，自动生成整体运营策略（Markdown 格式）
- **账号管理**：每客户下多个平台账号（抖音/小红书/视频号/快手/B站/TikTok）
  - 账号属性：名称、风格、核心关键词、目标定位
  - 支持编辑和删除
- **AI 账号规划方案**：基于客户信息 + 账号信息，生成账号级内容策略（Markdown 格式）

### 2. 数字人视频（创作工作台）

#### 项目管理
- 新建项目：弹窗选择客户 → 选择账号 → 创建项目
- 项目状态流转（单向，不可回退）：

| 状态 | 触发条件 | 触发方式 |
|------|---------|---------|
| 草稿 | 新建项目 | 系统自动 |
| 生产中 | 点击确认方案，AI 开始生成 | 系统自动 |
| 已完成 | AI 全部返回结果 | 系统自动 |
| 已发布 | 用户线下完成剪辑发布后回来标记 | 用户手动 |

- 标记已发布时可选填：发布时间、发布链接
- 发布信息在4处展示：生产层底部、创作工作台项目列表、社媒运营员近期项目、客户管理项目列表

#### 来源层（三选一）
1. 热点抓取：根据客户画像搜索热点话题
2. 甲方文案：粘贴或上传甲方原始文案
3. 深度调研：Gemini Deep Research 生成专业报告

最终输出一段素材文本。

#### 方案层
- 输入：素材文本 + 客户资料 + 账号信息 + 账号规划 + 业务结合方式 + 目标时长
- 业务结合方式：强关联 / 软植入 / 不结合
- 目标时长：15秒 / 30秒 / 45秒 / 60秒 / 90秒 / 120秒
- 输出 JSON：topic + style + tone + corePoints + scriptDraft（口播稿初稿）
- 用户审核编辑后确认，触发生产层

#### 生产层
- 项目主题字段（默认从方案层 AI 输出填充，可编辑）
- 布局：Tab 切换（非瀑布流）
- 4个 Tab **并行生成**，互不依赖，各自独立调用 AI：

| Tab | 输入依赖 | 输出格式 |
|-----|---------|---------|
| 口播稿 | 方案 JSON + 客户信息 | 纯文本 |
| 发布文案 | 方案 JSON + 平台规则 | JSON（title + body + tags） |
| 分镜头 | 方案 JSON + 时长 | JSON 数组（shot + duration + prompt + motion） |
| 素材图片 | 方案 JSON + 封面比例 | JSON 数组（purpose + ratio + prompt） |

- 每个 Tab 可单独点击「AI 重新生成」
- 数字人配置：选择数字人形象 + 选择音色
- 视频版本历史：支持切换查看不同版本

### 3. AI 创作

5个子功能，统一左栏表单 + 右栏预览墙布局：

#### 商品图生成（亚马逊7图）
- 上传1张产品图 → AI 自动识别商品信息 → 生成1-8张亚马逊规范图
- 产品信息：产品名称、一级类目（下拉，系统已有）、二级类目（文本输入，AI 填充）、首图语言、产品尺寸
- 生成设置：模型选择、图片比例、分辨率、生成张数
- 7张图类型：主图白底、多角度、场景图、尺寸标注、卖点图、细节特写、包装配件

#### 图生图
- 上传参考图 + 提示词 → AI 生成目标场景图片
- 支持「从提示词库选择」和「AI 生成提示词」
- 设置：模型、分辨率、图片比例、生图数量

#### 图生视频
- 上传1-2张参考图 + 分镜设置（2-5镜）+ 提示词 → AI 生成短视频
- DeepThink 功能：输入简单描述，AI 自动拆分镜头提示词
- 输出设置：宽高比、清晰度、去除原视频音频

#### 爆款视频复刻
- 上传爆款视频（或粘贴链接）→ AI 分析结构（镜头/节奏/转场）
- 上传产品图 → AI 用你的产品替换 → 生成同款结构新视频
- 参考：AdsTurbo Ad Clone

#### 提示词库
- 统一管理所有提示词，替代用户本地 txt 文档
- 三级数据隔离：系统预设（只读）/ 团队共享（管理员维护）/ 我的模版（个人）
- 支持分类、搜索、变量替换（如 `[Item]`）
- 嵌入所有创作工具的提示词输入框，一键插入

### 4. 社媒运营员（Agent 页面）

统计卡片：
- 已完成项目（本月增量）
- 已发布项目（发布率）
- Token 用量（本月消耗）
- 累计生成内容（口播稿+文案+图片总次数）

近期项目列表：客户名 · 项目主题 · 平台 + 状态标签

### 5. 其他模块
- 工作台首页（Dashboard）：数据概览和快捷入口
- 数据看板：运营数据统计（待开发）
- 系统设置：团队成员管理、角色权限（超管/管理员/成员）、角色下拉切换

## 提示词体系（共7套）

| 编号 | 名称 | 触发位置 | 输入 | 输出格式 |
|------|------|---------|------|---------|
| 1 | 客户运营策略 | 客户管理 → AI 运营方案 | 客户信息 + 平台列表 | Markdown |
| 2 | 账号规划方案 | 客户管理 → 账号 → AI 账号规划 | 客户信息 + 账号信息 + 平台特性 | Markdown |
| 3 | 方案层（视频方案初稿） | 创作工作台 → 方案层 | 素材文本 + 客户 + 账号 + 账号规划 + 业务结合 + 时长 | JSON |
| 4 | 口播稿精修 | 生产层 → 口播稿Tab | 方案 JSON + 客户信息 | 纯文本 |
| 5 | 发布文案 | 生产层 → 发布文案Tab | 方案 JSON + 平台规则 | JSON |
| 6 | 分镜头 | 生产层 → 分镜头Tab | 方案 JSON + 时长 | JSON 数组 |
| 7 | 素材图片 | 生产层 → 素材图片Tab | 方案 JSON + 封面比例 | JSON 数组 |

### 数据流

```
客户运营策略 → 指导整体方向
账号规划方案 → 指导单账号策略
     ↓
来源层 → 得到素材文本
     ↓
方案层（提示词3）→ { topic, style, tone, corePoints, scriptDraft }
     ↓ 用户确认
     ↓ 并行调用
  ┌──────┬──────┬──────┐
  ↓      ↓      ↓      ↓
提示词4  提示词5 提示词6 提示词7
口播稿   发布文案 分镜头  素材图片
```

## AI 模型选型

| 用途 | 模型 | 说明 |
|------|------|------|
| 文本生成（方案/文案/口播稿） | **Gemini 2.5 Flash**（主力）/ Gemini 2.5 Pro（高质量备选） | 中文强、支持 JSON 输出、成本低 |
| 图片生成/洗图 | **Nano Banana Pro**（= Gemini 3 Pro Image Preview） | 已有 API Key，多模态生图 |
| 产品信息提取 | Gemini 3 Pro Image Preview（多模态理解） | 上传产品图 → 返回产品名/类目/特征 JSON |
| 视频生成 | **可灵 V3 Omni** / Veo 3.1 / Vidu 2.0（选型中） | 见下方视频模型对比 |
| 洗视频 | 阿里云 Wan 2.6 VACE 或字节即梦（待定） | 推荐阿里云，有公开API |
| 商品介绍文案 | 其他团队开发中 | 后续接口接入 |

### API 调用关键配置

```python
generation_config = {
    "response_mime_type": "application/json",  # 强制输出合法 JSON
    "temperature": 0.7,                         # 方案层建议值
    "max_output_tokens": 4096
}
```

### 视频生成模型对比（选型中）

| | 可灵 V3 Omni | Veo 3.1 | Vidu 2.0 |
|---|---|---|---|
| 10秒视频价格 | ~$0.84（标准） | ~$1.50（Fast）/ ~$4.00（Standard） | ~$0.38 |
| 图生视频 | ✅ | ✅ | ✅ |
| 最高分辨率 | 4K | 1080p | 1080p |
| 中文理解 | 强 | 中 | 强 |
| 口型同步 | ✅ $0.07/5秒 | ❌ | ❌ |
| 适用场景 | 中文+数字人口型同步 | 最高画质品牌片 | 日常批量生产（最便宜） |

Temperature 建议值：
- 客户运营策略：0.5（稳定专业）
- 方案层：0.7（平衡创意和可控）
- 口播稿：0.8（活泼口语化）
- 发布文案：0.7（钩子感）
- 分镜头/素材图片：0.5（精确描述）

## 设计规范

- 所有组件使用 **inline styles**，不使用 Tailwind class
- 玻璃效果三件套：`background` (半透明白) + `backdropFilter` (blur+saturate) + `WebkitBackdropFilter`
- hover 效果通过 `onMouseEnter/onMouseLeave` 直接操作 style
- 动画使用 CSS `@keyframes` + inline `animation` 属性
- 图标统一用 lucide-react，strokeWidth 默认 1.7，active 态 2
- AI 创作模块统一布局：左栏表单（340px）+ 右栏预览墙（flex-1）
- 底部全宽蓝色渐变「立即生成」按钮
- 加载动画：蓝色圆环旋转 + "正在生成..."
